# 블로그 포스트 생성 가이드

## 구조 패턴 (상위 블로그 분석 기반)
- 제목: 40-55자. 숫자 포함 시 CTR +36% (예: "5가지 방법", "2026년 트렌드")
- 도입부 150자 이내: 문제 제기(38%) > 질문형(27%) > 통계형(20%) > 스토리형(15%)
- H2 소제목: 3-5개. 각 섹션 200-400자
- 총 분량: short=800-1200자, medium=1800-2500자, long=2800-4000자
- 마지막 섹션: 핵심 요약 2-3문장 + 행동 유도

## 단락 규칙
- 한 단락 2-4문장 (모바일 가독성 최적)
- 문장당 20-40자 (한국어 기준)
- 접속사로 시작하는 문장 20% 이하
- 능동태 > 수동태 (비율 7:3)

## SEO 패턴
- 핵심 키워드: 제목 앞부분 + 첫 H2 + 결론에 자연 배치
- 리스트/넘버링 포함 시 검색 노출 +28%
- 질문+답변 구조 포함 시 featured snippet 확률 상승
- 메타 설명용 요약: 120-155자 (검색결과 표시 최적)

## 금지 패턴
- "안녕하세요, 오늘은 ~에 대해 알아보겠습니다" (뻔한 인삿말)
- 과도한 이모지
- 근거 없는 수치 ("~의 90%가...")
- 동일 문장 구조 3회 연속 반복
- 소스 자료 원문 그대로 복사
