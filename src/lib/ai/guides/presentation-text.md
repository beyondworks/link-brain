# 발표용 텍스트 생성 가이드

## 구조 패턴
- 표지 슬라이드: 제목 (20자 이내) + 부제 + 발표자/날짜
- 목차 슬라이드: 3-5개 섹션 미리보기
- 본론 슬라이드: 섹션당 2-4장
- 요약 슬라이드: 핵심 3가지 recap
- Q&A / CTA 슬라이드

## 슬라이드 텍스트 규칙
- 제목: 한 줄, 15자 이내
- 핵심 메시지: 슬라이드당 1개 (50자 이내)
- 보조 불렛: 2-4개, 각 25자 이내
- "한 슬라이드 = 한 주장" 원칙

## 발표자 노트
- 각 슬라이드에 발표자 노트 포함 (구어체)
- 노트에 전환 멘트 포함 ("다음으로 넘어가면~")
- 예상 질문과 답변 1개씩

## 출력 형식
```
[슬라이드 N] 제목
핵심: (슬라이드에 표시할 텍스트)
- 불렛 1
- 불렛 2
[노트] 발표자가 말할 내용
```

## 금지 패턴
- 슬라이드에 문장형 텍스트 (읽어주기 발표)
- 불렛 6개 이상 (정보 과부하)
- 노트 없이 키워드만 나열
