# 인스타그램 피드 (캐러셀) 생성 가이드

## 구조 패턴 — 슬라이드 구성
- 슬라이드 1 (커버): 강렬한 제목 훅 (15자 이내) + 부제 (호기심 유발)
- 슬라이드 2-8 (본문): 한 슬라이드 = 한 포인트. 제목 + 2-3줄 설명
- 마지막 슬라이드: CTA ("저장하면 나중에 또 볼 수 있어요" / 팔로우 유도)

## 슬라이드별 텍스트 규칙
- 제목: 12자 이내 (한 눈에 읽히게)
- 본문: 슬라이드당 40자 이내 (3줄 한계)
- 넘버링 활용 ("1/7", "Step 1" 등)
- 출력 형식: [슬라이드 N] 제목 / 본문 으로 구분

## 캡션 규칙
- 첫 줄: 포스트 핵심 요약 (미리보기에 노출)
- 본문: 부가 설명 + 소스 언급
- 해시태그: 15-25개 (대/중/소 키워드 혼합)
- CTA: "저장", "공유", "댓글로 알려주세요"

## 톤
- 정보 전달형: 깔끔하고 단정한 문장
- 감성형: 짧은 문장, 여백 활용
- 교육형: 넘버링 + 한 문장 설명

## 금지 패턴
- 슬라이드에 긴 문단 (읽기 포기)
- 모든 슬라이드 동일 구조 반복
- 커버에 내용 전부 노출 (넘길 이유 없음)
