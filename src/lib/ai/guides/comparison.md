# 비교 분석 생성 가이드

## 구조 패턴 (의사결정 자료 분석)
- 비교 대상: 소스에서 2-4개 항목 식별
- 비교 축: 3-5개 기준 (성능, 비용, 사용성 등 소스 기반)
- 형식: 항목별 비교 → 종합 판단 순서
- 분량: short=항목당 2-3줄, medium=항목당 4-5줄, long=항목당 6-8줄

## 효과적 패턴
- 각 항목의 강점/약점 균형 있게 서술
- 수치 비교 시 동일 단위 통일
- 마지막에 "상황별 추천" 1-2문장

## 금지 패턴
- 소스에 없는 비교 항목 임의 추가
- 일방적 편향 (한쪽만 긍정/부정)
- 근거 없는 순위 매기기
