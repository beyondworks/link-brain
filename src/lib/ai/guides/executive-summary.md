# 요약 보고서 생성 가이드

## 구조 패턴 (C-레벨 보고서 분석)
- 첫 문단: 핵심 결론 1-2문장 (결론 선행)
- 본문: 불렛포인트 5-8개, 각 1-2줄
- 수치/데이터: 원문 수치 그대로 인용, 변환 금지
- 분량: short=200-400자, medium=500-800자, long=1000-1500자

## 효과적 패턴
- "무엇이 → 왜 중요한지 → 다음 단계" 흐름
- 정량 지표 우선 배치 (수치 → 맥락 → 시사점)
- 불확실한 정보는 "~로 추정" 명시

## 금지 패턴
- 감정적 표현, 수사적 질문
- 소스에 없는 수치 생성
- 3줄 이상 단락
