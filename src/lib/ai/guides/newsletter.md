# 뉴스레터 생성 가이드

## 제목 패턴 (오픈율 40%+ 기준)
- 길이: 25-40자
- 숫자 포함 → 오픈율 +22%
- 질문형 → 오픈율 +18%
- 이름 개인화 → 오픈율 +26%
- 괄호/대괄호 사용 → 오픈율 +15% (예: "[주간정리]", "(필독)")

## 본문 구조
- 인삿말: 1-2문장 (친근한 톤, 시의성 있는 화제)
- 핵심 콘텐츠: 3-5개 아이템, 각 100-200자 요약
- 각 아이템: 제목(볼드) + 요약 2-3문장 + "더 읽기" 링크
- CTA: 하단 1개 (명확한 행동 유도)
- 총 분량: 800-1500자 (읽기 시간 3-5분)

## 톤 규칙
- 편지체 (대화하듯 "~했어요", "~거든요")
- 전문 용어 사용 시 괄호로 설명 추가
- 이모지: 제목에 1개, 본문에 섹션당 0-1개
