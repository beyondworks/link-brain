# Threads 포스트 생성 가이드

## 구조 패턴
- 500자 이내 단일 포스트 (스레드 분할 지양)
- 첫 줄: 훅 (질문/반전/숫자/대담한 주장) — 스크롤 멈추게 하는 한 줄
- 2-4줄: 핵심 메시지 (1가지 주제만)
- 마지막: CTA 또는 여운 (질문형 마무리 댓글 유도)

## 줄바꿈 규칙
- 문장마다 줄바꿈 (벽 텍스트 금지)
- 빈 줄로 호흡 조절 (2-3문장마다)
- 들여쓰기/리스트 마커 사용 가능 (→, -, ·)

## 해시태그
- 본문 끝에 3-5개
- 대주제 1 + 세부주제 2-3 + 트렌드 1
- 해시태그를 문장 안에 섞지 말 것

## 톤 기준
- 구어체 (글말투 X)
- "~합니다" 보다 "~거든요", "~더라고요"
- 개인 경험 프레이밍 ("저는 이렇게 봐요")

## 금지 패턴
- 뻔한 명언 인용으로 시작
- "오늘은 ~에 대해 이야기해볼게요"
- 이모지 5개 이상
- 여러 주제를 한 포스트에 욱여넣기
