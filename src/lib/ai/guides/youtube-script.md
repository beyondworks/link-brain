# 유튜브 대본 생성 가이드

## 구조 패턴
- 인트로 (0:00-0:15): 훅 — 질문/충격/약속 ("이걸 모르면 손해입니다")
- 브릿지 (0:15-0:30): 영상 미리보기 ("오늘 3가지를 알려드릴게요")
- 본론: 챕터별 구분 (각 2-5분)
- 아웃트로: 요약 + CTA ("구독/좋아요/댓글")

## 본론 챕터 규칙
- 챕터당 하나의 핵심 주제
- 챕터 시작마다 타임스탬프 표기 ([00:00])
- 챕터 전환: "자, 그럼 다음으로~" (명시적 전환)
- 각 챕터: 주장 → 근거/예시 → 정리

## 대본 작성 규칙
- 구어체 필수 ("~거든요", "~잖아요", "~인데요")
- 문장 길이: 20-35자 (한 호흡에 읽을 수 있게)
- 강조 표기: *중요한 부분* (편집 포인트)
- 시청자 호출: "여러분", "혹시 ~해보신 적 있으세요?"

## 출력 형식
```
[00:00] 인트로
(대사)

[00:15] 브릿지
(대사)

[01:00] 챕터 1: 제목
(대사)
```

## 인트로 훅 패턴 (효과순)
1. 결론 선행: "결론부터 말하면 ~입니다"
2. 반전: "~라고 생각하시죠? 사실은 ~"
3. 숫자: "3가지만 알면 됩니다"
4. 공감: "저도 처음엔 ~했어요"

## 금지 패턴
- "안녕하세요 오늘은~" 으로 시작 (이탈률 최고)
- 문어체 ("~하였습니다", "~됩니다")
- 한 챕터에 여러 주제 혼합
- CTA 없는 아웃트로
