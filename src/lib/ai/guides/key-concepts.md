# 핵심 포인트 생성 가이드

## 구조 패턴 (학습 자료 분석)
- 개수: 5-7개 핵심 개념 추출
- 각 항목: 개념명(한 줄) + 설명 1-2문장(40-80자)
- 순서: 기본 → 심화 (난이도 순) 또는 중요도 순
- 분량: short=5개, medium=7개, long=10개

## 효과적 패턴
- 개념 간 관계 언급 ("A는 B의 전제 조건")
- 비유/예시 1개씩 포함 시 이해도 +40%
- 원문 용어 그대로 사용 (임의 번역 금지)

## 금지 패턴
- 소스에 없는 개념 추가
- 동일 수준의 내용 중복 나열
- 설명 없는 용어 나열
